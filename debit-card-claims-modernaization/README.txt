# Debit Card Claims Modernization

Modern open-source replacement architecture for PEGA-based debit card claims processing.

## Goals
- Claim + dispute lifecycle management
- Workflow orchestration
- AI-assisted investigator support
- Provisional credit decision assistance
- Modern API-first architecture

## Planned Stack
- FastAPI
- PostgreSQL
- LangChain
- Ollama
- Temporal/Camunda (future)
